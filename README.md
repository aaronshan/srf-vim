### 如何使用

#### 1. 安装
打开客户端，执行:
```
wget -qO- https://github.com/aaronshan/srf-vim.git/master/install.sh | sh -x
```

#### 2. 卸载

打开客户端，执行:
```
wget -qO- https://github.com/aaronshan/srf-vim.git/master/uninstall.sh | sh -x
```
